# 📚 BookList SPA

Aplicación web desarrollada con **Vue.js 3** que permite gestionar un catálogo de libros mediante una interfaz interactiva tipo **SPA (Single Page Application)**.

## 🚀 Tecnologías utilizadas

* Vue.js 3
* Vite
* Vue Router
* JavaScript
* HTML5
* CSS3

## ⚙️ Instalación y ejecución

1. Clonar o descargar el proyecto.

2. Instalar las dependencias:

npm install

3. Ejecutar el servidor de desarrollo:

npm run dev

4. Abrir en el navegador:

http://localhost:5173

## 📌 Funcionalidades

* Agregar nuevos libros mediante formulario reactivo (`v-model`)
* Visualizar libros en una lista dinámica (`v-for`)
* Eliminar libros mediante eventos (`@click`)
* Filtrar libros por autor o categoría
* Navegación entre vistas con **Vue Router**
* Visualización del detalle de cada libro con **rutas dinámicas**

## 🗂️ Estructura del proyecto

src/

* components → Componentes reutilizables
* views → Vistas principales
* router → Configuración de rutas
* data → Almacenamiento reactivo
* App.vue → Componente principal
* main.js → Punto de entrada
